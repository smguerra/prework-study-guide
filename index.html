Prework Study Guide

HTML

  • The charset element sets out character encoding.
  • Elements nested inside other elements are called child elements.
  • Elements that contain child elements are called parent elements.
  • img elements contain information about images that are displayed on the webpage.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • Class attributes allow us to share a CSS rule to any element we choose.
  • Every each HTML element is essentially a box.
  • There are three ways to style webpages, inline CSS and internal and external CSS style sheets.
  • Separation of concerns: separating code into different files based on use.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git checkout main: used to switch to the main branch
  • git pull origin main: used to pull down the latest main branch
  • git push origin: to push changes.
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
