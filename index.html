HTML

  • The charset element sets out character encoding.
  • Elements nested inside other elements are called child elements.
  • Elements that contain child elements are called parent elements.
  • img elements contain information about images that are displayed on the webpage.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • Class attributes allow us to share a CSS rule to any element we choose.
  • Every each HTML element is essentially a box.
  • There are three ways to style webpages, inline CSS and internal and external CSS style sheets.
  • Separation of concerns: separating code into different files based on use.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git checkout main: used to switch to the main branch
  • git pull origin main: used to pull down the latest main branch
  • git push origin: to push changes.
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • JavaScript is a high-level programming language.
  • JavaScript allows users to interact with web-sites.
  • Functions are the building blocks of JavaScript.
  • Console.log is used to output a message to the web-console by adding an argument.
  • if statements are a way of introducing decision-making in our code.
  • Array is a group of related data.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
